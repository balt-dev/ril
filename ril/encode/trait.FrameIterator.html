Expand description

Represents the lazy decoding of frames from an encoded image sequence, such as an animated image.

See Also

  • ImageSequence
  • Frame

Required Methods

source

fn len(&self) -> u32

Returns the number of frames in the sequence.

This does not consume any frames as this data is usually known from the very beginning of decoding.

source

fn loop_count(&self) -> LoopCount

Returns the amount of times this sequence will loop over itself.

Provided Methods

source

fn is_empty(&self) -> bool

Returns if there are no frames in the sequence. In this case, the image is probably invalid to be encoded again.

source

fn into_sequence(self) -> Result<ImageSequence<P>>where
    Self: Sized,

Collects all frames in this iterator and turns it into a high level ImageSequence. If any frame fails, that error is returned.

Errors
